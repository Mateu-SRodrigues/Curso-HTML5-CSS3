Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> </b>(NÃO SEMÂNTICA, TEM SOMENTE FORMA)

Nesta frase tenho um termo em destaque usando a tag <strong> </strong>(SEMÂNTICA, TEM FORMA E SIGNIFICADO)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> </i>(NÃO SEMÂNTICA)

Nesta frase temos um termo em ênfase usando a tag <em> </em>(SEMÂNTICA)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> </mark>(SEMÂNTICO)

Texto grande e pequeno

Estamos criando um texto grande usando a tag <big> </big>(SOMENTE FORMA) e um texto pequeno usando a tag <small> </small> nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído usando a tag <del> </del>, para indicar que ele deve ser lido,mas não considerado

Texto inserido

Podemos marcar um texto como inserido usando a tag <ins> </ins>(COM SIGNIFICADO), para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u> </u>(SEM SIGNIFICADO)
